# 🎓 Programa de Desenvolvedor Fullstack - SENAC

Este repositório reúne os projetos, exercícios e códigos práticos desenvolvidos por mim durante o **Programa de Desenvolvedor Fullstack do SENAC**. 

O objetivo deste espaço é registrar minha evolução técnica e demonstrar a aplicação prática de conceitos de desenvolvimento front-end, lógica de programação, interatividade e design de interface (UI/UX) para os avaliadores do **WebAcademy (UFAM)**.

---

## 🚀 Visão Geral do Aprendizado & Competências

O curso foi estruturado em Unidades Curriculares (UCs) focadas em competências reais de mercado. Abaixo, destaco as habilidades validadas no meu histórico (com aproveitamento máximo - Menção **A**):

### 1. Codificação de Interfaces Web (HTML5 & CSS3)
*   **Foco:** Construção de páginas modernas, semânticas e responsivas.
*   **Competências:** Elaboração de layouts a partir de propostas gráficas, análise rigorosa de **usabilidade e acessibilidade** em desktops e dispositivos móveis.

### 2. Interatividade Web (JavaScript ES6)
*   **Foco:** Dar vida às páginas e criar experiências dinâmicas para o usuário.
*   **Competências:** Lógica de programação aplicada (variáveis, estruturas de decisão, laços de repetição e funções) e validação de interatividade para garantir uma experiência livre de bugs.

### 3. Implantação e Boas Práticas (DevOps & Codificação)
*   **Foco:** Escrever código limpo, depuração eficiente e publicação de aplicações.
*   **Competências:** Seleção de metodologias de codificação, adoção de **melhores práticas** (Clean Code) e resolução rápida de erros (*debugging*).

### 4. Design de Interface & Tratamento de Imagem (Adobe Photoshop)
*   **Foco:** Preparação visual e layout focado na experiência do usuário.
*   **Competências:** Tratamento e otimização de imagens para a web e planejamento de layouts baseado nos fundamentos de **Arquitetura de Informação**.
